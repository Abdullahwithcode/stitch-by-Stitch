# Stitch by Stitch — CMT Garment Manufacturing Website

A clean, static website for Stitch by Stitch (CMT garment manufacturing) built with HTML, CSS, and minimal JS. It presents services, facilities, contact info, and a gallery — optimized for clarity and quick navigation.

## Features
- Modern blue theme (primary `#3B82F6`, hover `#1e293b`)
- Centered navbar links with active underline (CSS-only)
- Dedicated pages: About, Services, Facilities, Contact, Gallery
- Single-line timelines and standardized grids for clarity
- Footer-themed CTAs added above footers across pages for conversion
- Font Awesome 6.4 via CDN for icons
